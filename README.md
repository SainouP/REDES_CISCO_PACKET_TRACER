##  Proyecto de Infraestructura de Red (Cisco)

Diseño e implementación de una infraestructura de red multi-sede, asegurando una comunicación confiable entre las sedes (Lima, Piura, Arequipa, Cajamarca, Cusco).

###  Miembros
- Alvaro Anchante
- Rodrigo Jara
- Carlos Cardoza
- Roseline bances

###  Características principales
- Diseño de red LAN/WAN
- Direccionamiento IP y subneteo
- Segmentación mediante VLAN y enrutamiento InterVLAN
- Configuración de DHCP y DNS
- Configuración de routers y switches
- Implementación de redes inalámbricas (WiFi)
- Aplicación de políticas de seguridad

###  Tecnologías
- Cisco Packet Tracer
